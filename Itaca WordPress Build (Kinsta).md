# Itaca WordPress Build (Kinsta)
Performance-first WordPress build using:
- Kinsta Hosting (Edge Caching + CDN)
- Blocksy Pro (Business)
- Elementor Pro
- ACF Pro (JSON fields included)
- Perfmatters Performance Plugin
- Fluent Forms Pro

Environment: PHP 8.2, WP 6.8.3
GA4/GTM implemented via functions.php snippet.

This repo is a *read-only mirror* for AI-assisted analysis, planning, and snippet development.
Do not push directly to production.
